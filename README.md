# Dr. Thota Sivasankar - Personal Academic Website

This is a ready-to-upload GitHub Pages website.

## How to publish
1. Create a GitHub repository named `thotasivasankar.github.io` or `siva-iirs.github.io`.
2. Upload all files and folders from this package.
3. Go to Repository Settings → Pages.
4. Select branch: `main`, folder: `/root`.
5. Save. Your website will be live within a few minutes.

## How to update photo
Replace `assets/img/profile.jpg` with your professional photo and update the image tag in `index.html` if needed.
